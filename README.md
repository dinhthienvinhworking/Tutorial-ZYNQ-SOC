# Tutorial 1 - First Zynq Design

Môn: Thiết kế kết hợp HW/SW

## Nội dung

- Tạo Zynq Processing System
- Kết nối AXI GPIO
- Điều khiển LED qua phần mềm chạy trên ARM

## Cấu trúc

- `vivado_hw/`: project Vivado
- `vitis_ws/`: workspace Vitis
- `export/`: file XSA export từ Vivado
- `docs/`: ghi chú/tài liệu
